## Usage

Badge has 3 variants:

- `light` (default) – light background, dark color, no border
- `outline` – transparent background, dark color and border
- `filled` – dark background, white color, no border

Default Badge color is `theme.primaryColor`. You can choose any color defined in <GatsbyLink to="/pages/theming/">theme</GatsbyLink>.

## Size

Badge has predefined sizes: xs, sm, md, lg, xl.
Size prop controls badge height, padding and font-size.

```tsx
<Badge size="sm" /> // -> predefined small size
<Badge size="xl" /> // -> predefined extra large size
```

Badge sizes from xl to xs:

<BadgeSizesDemo />

You can get predefined badge heights by importing `BADGE_SIZES`:

```tsx
import { BADGE_SIZES } from '@mantine/core';
```

## Radius

xs, sm, md, lg, xl radius values are defined in <GatsbyLink to="/pages/theming/">theme.radius</GatsbyLink>.
Alternatively you can use number to set radius in px:

```tsx
<Badge radius="lg" /> // -> theme predefined large radius
<Badge radius={1000} /> // -> { borderRadius: 1000 }
```

## Full width and overflow

Badge can take full width of container if you pass `fullWidth` prop.
If badge is too large for its container, overflow content will be hidden with ellipsis.
