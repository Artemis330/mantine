## Get step ref

You can get refs of step button and stepper root element (div):

```tsx
import { useRef } from 'react';
import { Stepper } from '@mantine/core';

function MyStepper() {
  const firstStep = useRef();
  const stepper = useRef();

  return (
    <Stepper ref={stepper}>
      <Stepper.Step label="First step" ref={firstStep} />
      <Stepper.Step label="Second step" />
    </Stepper>
  );
}
```

## Accessibility

`<Stepper.Step />` components render button element, set `aria-label` or `title` props
to make component visible for screen readers in case you do not specify `label` or `description`:

```tsx
<Stepper.Step /> // -> not ok, empty labels for screen reader
<Stepper.Step label="Step 1" description="Create an account" /> // -> ok
<Stepper.Step aria-label="Create an account" /> // -> ok
<Stepper.Step title="Create an account" /> // -> ok
```

## TypeScript

### Component props

Import component props type:

```tsx
import type { StepperProps, StepProps } from '@mantine/core';
```

- `StepperProps` extends `React.ComponentPropsWithoutRef<'div'>`
- `StepProps` extends `React.ComponentPropsWithoutRef<'button'>`

### StepperNames type

`StepperNames` type is a union with Stepper [Styles API](/theming/styles-api/) names:

```tsx
import type { StepperNames } from '@mantine/core';

const stepperClasses: Partial<Record<StepperNames, 'string'>> = {
  step: 'stepper-step',
  stepCompleted: 'stepper-completed',
};

<Stepper classNames={stepperClasses} />;
```

See full styles descriptions list under _Styles API_ tab.
