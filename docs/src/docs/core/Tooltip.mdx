## Multiline

By default, tooltip white-space property is set to nowrap. To change that set:

- **multiline** prop to enable line breaks
- **width** prop to set tooltip width in px

<Demo data={TooltipDemos.multiline} />

## Change transition

Tooltip is built with [Transition](/core/transition/) component, it supports the following props:

- **transition** – premade transition name or transition object
- **transitionDuration** – transition duration in ms, defaults to 100ms

```tsx
<Tooltip transition="skew-up" transitionDuration={300} transitionTimingFunction="ease" />
```

All available predefined transitions:

<Demo data={TooltipDemos.transitions} />

## Close and open delay

You can delay tooltip open/close events by setting `openDelay` and `closeDelay` props in ms.
Both props default to 0 and reset if user moves mouse over/out of target element before delay is expired:
