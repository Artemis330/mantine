## TypeScript

@mantine/core package exports following types associated with Breadcrumbs:

- `BreadcrumbsProps` – Breadcrumbs component props, extends `React.ComponentProps<'div'>`
- `BreadcrumbsStylesNames` – a union of styles names that can be used with [Styles API](/theming/styles-api/) (`classNames` and `styles` props)
