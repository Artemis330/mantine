## Change color and size

Burger has predefined sizes: xs, sm, md, lg, xl.
You can pass size as number and it will be set as width and height in px.

```tsx
<Burger size="sm" /> // -> predefined small size
<Burger size={50} /> // -> { width: 50, height: 50 }
```

### Default size values

You can get default sizes for burger by importing `BURGER_SIZES`:

```tsx
import { BURGER_SIZES } from '@mantine/core';
```

## Get element ref

You can get button ref by passing `elementRef` prop to Burger component:

```tsx
import React, { useRef } from 'react';
import { Burger } from '@mantine/core';

function BurgerDemo() {
  const ref = useRef();
  return <Burger elementRef={ref} opened />;
}
```
