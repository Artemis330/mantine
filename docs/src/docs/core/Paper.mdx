## Usage

Paper component renders white (or `theme.colors.dark[7]` for dark theme)
background with shadow, border-radius and padding from theme.

<PaperConfigurator />

## Shadow

You can use predefined shadows defined in <GatsbyLink to="/pages/theming/">theme</GatsbyLink> or your own:

<PaperShadowDemo />

## Padding

Paper has predefined padding: xs, sm, md, lg, xl defined in <GatsbyLink to="/pages/theming/">theme.spacing</GatsbyLink>.
Alternatively you can use number to set padding in px:

```tsx
<Paper size="sm" /> // -> theme predefined small spacing
<Paper size={20} /> // -> { padding: 20 }
```

## Radius

xs, sm, md, lg, xl radius values are defined in <GatsbyLink to="/pages/theming/">theme.radius</GatsbyLink>.
Alternatively you can use number to set radius in px:

```tsx
<Paper radius="lg" /> // -> theme predefined large radius
<Paper radius={0} /> // -> { borderRadius: 0 }
```

## Get element ref

You can get root element ref with `elementRef` prop:

```tsx
import React, { useRef } from 'react';
import { Paper } from '@mantine/core';

function Demo() {
  const paperRef = useRef();
  return <Paper elementRef={paperRef} />;
}
```
