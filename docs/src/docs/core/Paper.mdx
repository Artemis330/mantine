## Padding and radius

Paper has predefined padding: xs, sm, md, lg, xl defined in <GatsbyLink to="/pages/theming/">theme</GatsbyLink>.
You can pass padding as number and it will be set as padding on all sides in px.

```tsx
<Paper size="sm" /> // -> theme predefined small spacing
<Paper size={20} /> // -> { padding: 20 }
```

Same with radius â€“ predefined values for xs, sm, md, lg, xl are defined in <GatsbyLink to="/pages/theming/">theme</GatsbyLink>,
or pass in number to set border-radius in px:

```tsx
<Paper radius="lg" /> // -> theme predefined large radius
<Paper radius={0} /> // -> { borderRadius: 0 }
```

## Get element ref

You can get button ref by passing `elementRef` prop to Paper component:

```tsx
import React, { useRef } from 'react';
import { Paper } from '@mantine/core';

function PaperDemo() {
  const paperRef = useRef();
  return <Paper elementRef={paperRef} />;
}
```
