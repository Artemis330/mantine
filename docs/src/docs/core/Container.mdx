# Container

Container is the basic layout element, it centers content horizontally and add horizontal padding from theme.

<PageHeader
  source="mantine-core/src/Container/Container.tsx"
  docs="core/Container.mdx"
  package="@mantine/core"
/>

```tsx
import { Container } from '@mantine/core';
```

## Usage

Container accepts props:

- `size` – controls default max width
- `padding` – controls horizontal padding of container, pass `xs | sm | md | lg | xl` for value defined in `theme.spacing` or number to set horizontal padding in px
- `fluid` – overwrites `size` and sets max width to 100%
