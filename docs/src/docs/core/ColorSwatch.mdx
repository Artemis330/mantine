## Usage

Use ColorSwatch to display color

```tsx
<ColorSwatch color="#c3c3c3" />
```

<ColorSwatchBaseDemo />

## Size and radius

ColorSwatch does not have predefined size values, pass number value to set width and height in px:

```tsx
<ColorSwatch size={50} /> // -> { width: 50, height: 50 }
```

Radius has predefined values for xs, sm, md, lg, xl which are defined in <GatsbyLink to="/theming">theme</GatsbyLink>,
or you can pass in number to set border-radius in px:

```tsx
<ColorSwatch radius="lg" /> // -> theme predefined large radius
<ColorSwatch radius={10} /> // -> { borderRadius: 10 }
```
