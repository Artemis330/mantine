## Usage

Use Highlight component to highlight part in given string with mark tag.

Pass main string as children to Highlight component and string part that should be highlighted to `highlight` prop.
If main string does not include `highlight` part, it will be ignored.
Component ignores trailing whitespace and highlights only first matched characters sequence.

You can change color of mark by passing `highlightColor` prop with <GatsbyLink to="/pages/theming/">theme</GatsbyLink> color.
