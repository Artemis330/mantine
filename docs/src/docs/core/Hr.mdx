## Sizes

Hr has predefined sizes: xs, sm, md, lg, xl.
You can pass size as number and it will be set as width and height in px.

```tsx
<Hr size="sm" /> // -> predefined small size
<Hr size={50} /> // -> { width: 50, height: 50 }
```

### Default size values

You can get default sizes for action icon by importing `HR_SIZES`:
