## Sizes

Hr has predefined sizes: xs, sm, md, lg, xl.
You can pass size as number and it will be set as height in px:

```tsx
<Hr size="sm" /> // -> predefined small size
<Hr size={50} /> // -> { width: 50, height: 50 }
```

You can get predefined sizes by importing `HR_SIZES`:

```tsx
import { HR_SIZES } from '@mantine/core';
```
