# InnerHtml

Render html from string with styles from mantine theme

```tsx
import { InnerHtml } from '@mantine/core';
```

## Usage

InnerHtml sets given html string to `innerHtml` of div element.
Html has all styles from mantine theme as all other components.
**Warning:** when working with user input consider [purifying](https://github.com/cure53/DOMPurify) html string to avoid XSS attacks
