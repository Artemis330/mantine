## Usage

Switch is another variant of <GatsbyLink to="/core/Checkbox">Checkbox</GatsbyLink>
component. Use it to capture boolean value input from user.
For better accessibility set `label` prop, it will add associated label element.

## Sizes

Switch has 5 predefined sizes: xs, sm, md, lg, xl.
Size defines label font-size (from <GatsbyLink to="/pages/theming/">theme.fontSizes</GatsbyLink>), input width and height.

<SwitchSizesDemo />

You can get switch sizes by importing `SWITCH_SIZES`:

```tsx
import { SWITCH_SIZES } from '@mantine/core';
```

## Radius

Radius controls border-radius of body and handle.
xs, sm, md, lg, xl radius values are defined in <GatsbyLink to="/pages/theming/">theme.radius</GatsbyLink>.
Alternatively you can use number to set radius in px:

```tsx
<Switch radius="lg" /> // -> theme predefined large radius
<Switch radius={10} /> // -> { borderRadius: 10 }
```

## Controlled

```tsx
import React, { useState } from 'react';
import { Switch } from '@mantine/core';

function Demo() {
  const [checked, setChecked] = useState(false);
  return <Switch checked={checked} onChange={(event) => setChecked(event.currentTarget.checked)} />;
}
```

## Get element ref

You can get input ref with `elementRef` prop to:

```tsx
import React, { useRef } from 'react';
import { Switch } from '@mantine/core';

function Demo() {
  const ref = useRef();
  return <Switch elementRef={ref} />;
}
```

## Accessibility

Switch is a regular input with checkbox type. Provide `aria-label` in case you use switch without label for screen reader support:

```tsx
<Switch label="My switch" />; // -> ok, input and label is connected
<Switch />; // not ok, input is not labeled
<Switch aria-label="My switch" />; // -> ok, label is not visible but will be announced by screen reader
```
