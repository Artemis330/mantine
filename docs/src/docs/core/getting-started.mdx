# Mantine core

@mantine/core is the main package that includes all essential components
that you may need to build an application.

## Installation

With yarn:

```sh
yarn add react-jss @mantine/theme @mantine/hooks @mantine/core
```

With npm:

```sh
npm install react-jss @mantine/theme @mantine/hooks @mantine/core
```

With [peerdeps](https://www.npmjs.com/package/install-peerdeps):

```sh
npx install-peerdeps @mantine/core
```
