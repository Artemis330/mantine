# Getting started

@mantine/core is the main package that includes all essential components
that you may need to build an application.

## Installation

With yarn:

```sh
yarn add react-jss @mantine/theme @mantine/hooks @mantine/core @modulz/radix-icons
```

With npm:

```sh
npm install react-jss @mantine/theme @mantine/hooks @mantine/core @modulz/radix-icons
```

With peerdeps:

```sh
npx install-peerdeps @mantine/core
```
