# Getting started

@mantine/core is the main package that includes all essential components
that you may need to build an application.

## Installation

With yarn:

```sh
yarn add @mantine/theme @mantine/hooks @mantine/core react-jss
```

With npm:

```sh
npm install @mantine/theme @mantine/hooks @mantine/core react-jss
```

## Dependencies

To use @mantine/core you will need React version 16.8.0 or above.
