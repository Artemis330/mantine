## Usage

Use Loader to indicate loading state.

By default loader will be rendered with `theme.primaryColor`.
You can choose any color defined in <GatsbyLink to="/pages/theming/#colors">theme.colors</GatsbyLink>:

<LoaderConfigurator />

## Size

Loader has predefined sizes: xs, sm, md, lg, xl. Size controls loader height.
Alternatively you can use number to set height in px:

```tsx
<Loader size="sm" /> // -> predefined small size
<Loader size={50} /> // -> { height: 50 }
```

Predefined sizes from xl to xs:

<LoaderSizesDemo />

You can get predefined sizes by importing `LOADER_SIZES`:

import { LOADER_SIZES } from '@mantine/core';
