# Text

Render text and links with theme styles

<PageHeader source="mantine-core/src/Text/Text.tsx" docs="core/Text.mdx" package="@mantine/core" />

```tsx
import { Text } from '@mantine/core';
```

## Usage

Use Text component to render text and links with theme styles. Control Text styles with props:

- `size` – predefined font-size from `theme.fontSizes`
- `color` – predefined colors from `theme.colors`
- `weight` – font-weight property
- `transform` – text-transform property
- `align` – text-align property
- `variant` – text or link predefined styles

<TextBaseDemo />

## Custom component

By default text is rendered in div element, you can change it by setting `component` prop:

```tsx
import { Link } from 'react-router-dom';

<Text variant="link" component={Link} to="/hello">
  React router link
</Text>

<Text variant="link" component="a" href="https://mantine.dev">
  Normal link
</Text>
```
