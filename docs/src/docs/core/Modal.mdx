## Usage with Portal

Usually, it is a good idea to render modal outside of current component to avoid styles collisions.
To implement this use <GatsbyLink to="/core/portal">Portal</GatsbyLink> component.

```tsx
import React, { useState } from 'react';
import { Modal, Portal } from '@mantine/core';

function PortalDemo() {
  const [opened, setOpened] = useState(false);

  return (
    <div style={{ position: 'relative', zIndex: 1 }}>
      <Portal zIndex={5}>
        <Modal opened={opened} onClose={() => setOpened(false)}>
          Modal is rendered at the end of document.body
        </Modal>
      </Portal>

      <button onClick={() => setOpened(true)}>Open modal</button>
    </div>
  );
}
```
