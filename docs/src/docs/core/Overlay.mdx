## Usage

Overlay had absolute position and will take 100% of width and height of parent container.
It is used to build components like <GatsbyLink to="/core/modal">Modal</GatsbyLink> and <GatsbyLink to="/core/loading-overlay">LoadingOverlay</GatsbyLink>.

You can change overlay opacity (from 0 to 1) and color (css color value, not connected to mantine theme).
