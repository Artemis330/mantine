## Usage

Card component is a wrapper around <GatsbyLink to="/core/paper/">Paper</GatsbyLink> component
with context styles for <GatsbyLink to="/core/image/">Image</GatsbyLink> and <GatsbyLink to="/core/hr/">Hr</GatsbyLink>
components. Combine Card component with other Mantine components to create product cards:

<CardBaseDemo />

## Link overlay

Use <GatsbyLink to="/core/overlay/">Overlay</GatsbyLink> component with 0 opacity
to create link overlay. Note that, card context styles (negative margins and border-radius) rely on `:fist-child` and `:last-child`,
if you need preserve them, do not put overlay as first or last element.

<CardOverlayDemo />

## Get element ref

You can get root element ref by setting `elementRef` prop:

```tsx
import React, { useRef } from 'react';
import { Card } from '@mantine/core';

function Demo() {
  const ref = useRef();
  return <Card elementRef={ref} />;
}
```

## CardsGrid

Use CardsGrid component to display grid of cards:

- Control amount of cards per row with `cardsPerRow` prop
- Set `gutter` with xs, sm, md (default), lg or xl value to set spacing between cards with `theme.spacing` value or set `gutter` as number to set spacing in px

<CardsGridDemo />

Set `grow` prop to make items at the last row take all space:

<CardsGridGrowDemo />

Combine CardsGrid with <GatsbyLink to="/hooks/use-media-query">use-media-query</GatsbyLink> hook to make amount of cards responsive.
In this example:

- viewport >= 1000px -> 4 cards per row
- 1000px > viewport >= 750px -> 3 cards per row
- 750px > viewport >= 540px -> 2 cards per row
- 540px > viewport -> 1 card per row

Try resizing browser to see cards per row change:

<CardsGridBreakpointsDemo />
