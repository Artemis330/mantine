## Color and title props

You can choose any color defined in <GatsbyLink to="/theming">theme</GatsbyLink>. By default Alert will have `theme.primaryColor` color.

```tsx
<Alert /> // -> theme.primaryColor (defaults to blue)
<Alert color="red" /> // -> red color
```

If you pass `title` prop high contrast title will be rendered.

```tsx
<Alert title="Bummer!" /> // -> with title
<Alert /> // -> no title, just body
```
