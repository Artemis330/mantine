## Colors

ActionIcon `color` prop accepts only keys of `theme.colors`, hex, rgba, hsl and other CSS colors are not supported:

<Demo data={ActionIconDemos.colors} />

## Close button

CloseButton is a premade ActionIcon with close icon, it is used in all other components: [Popover](/core/popover/), [Modal](/core/modal/), [Notification](/core/Notification/), etc.
Component accepts the same props as ActionIcon with additional `iconSize` prop to control icon width and height:

<Demo data={ActionIconDemos.closeButton} />

## Change root element

```tsx
import { ActionIcon } from '@mantine/core';

function Demo() {
  return (
    <ActionIcon component="a" href="https://mantine.dev/core/">
      Anchor element
    </ActionIcon>
  );
}
```

## Usage with react-router

```tsx
import { Link } from 'react-router-dom';
import { ActionIcon } from '@mantine/core';

function Demo() {
  return (
    <ActionIcon component={Link} to="/react-router">
      React router link
    </ActionIcon>
  );
}
```

## Get element ref

```tsx
import { useRef } from 'react';
import { ActionIcon } from '@mantine/core';

function Demo() {
  const ref = useRef<HTMLButtonElement>();
  return <ActionIcon ref={ref} />;
}
```

## Accessibility

ActionIcon does not have an associated label, set `title` or `aria-label` props to make it visible for screen readers:

```tsx
<ActionIcon /> // -> not visible to screen reader
<ActionIcon title="Settings" /> // -> ok
<ActionIcon aria-label="Settings" /> // -> ok
```
