## Usage

use-scroll-into-view handles scroll behavior over any scrollable element.
Basic usage works just as `element.scrollIntoView()`. 

It allows you adjust the scrolling animation with respect to the `reduced-motion` user preference.

## API

Hook accepts 5 arguments:

- `onScrollFinish` – optional function that will be called after scroll animation
- `easing` – optional custom math easing function
- `duration` - optional duration of scroll animation
- `axis` - optional axis of scroll
- `cancelable` - optional indicator if animation may be interrupted by user scrolling
- `offset` - additional distance between nearest edge and element

Hook returns object with 4 elements methods `scrollIntoView` and `cancel`.

- `scrollIntoView` – function that starts scroll animation
- `cancel` – function that stops scroll animation
- `targetRef` - ref of target HTML node
- `scrollableRef` - ref of scrollable parent HTML element, if not used document element will be used

`scrollIntoView` method accepts 1 argument:

- `alignment` - optional target element alignment relatively to parent based on current axis

```tsx
scrollIntoView({
  alignment: 'center'
})
```

## Easing

Hook accept custom `easing` math function to control the flow of animation.
It takes `t` argument which is a number between `0` and `1`.

Default easing is `easeInOutQuad` - more info [here](https://easings.net/#easeInOutQuad).
You can find popular easings examples on [easings.net](https://easings.net/)

```tsx
useScrollIntoView({
  target: someDomElement,
  easing: (t) => t < 0.5 ? 16 * Math.pow(t, 5) : 1 - Math.pow(-2 * x + 2, 5) / 2, // easeInOutQuint
});
```

## Definition

```tsx
function useScrollIntoView<
  Target extends HTMLElement,
  Parent extends HTMLElement | null = null
>({
  onScrollFinish?: () => void;
  duration?: number;
  axis?: 'x' | 'y';
  easing?: (t: number) => number;
  offset?: number;
  cancelable?: boolean;
}): { 
  targetRef: MutableRefObject<Target>;
  scrollableRef: MutableRefObject<Parent>;
  scrollIntoView: ({
    alignment?: 'start' | 'end' | 'center';
  }) => void;
  cancel: () => void;
};
```
