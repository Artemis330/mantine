## Usage

use-did-update is a wrapper for common updates tracking pattern:

```tsx
const mounted = useRef(false);

useEffect(() => {
  if (mounted.current) {
    fn();
  } else {
    mounted.current = true;
  }
}, dependencies);
```

This prevents fn function call in useEffect on initial mount:

```tsx
useDidUpdate(() => console.log("Won't be called when mounted"), [value]);
```

## Definition

```tsx
function useDidUpdate(fn: () => void, dependencies?: any[]): void;
```
