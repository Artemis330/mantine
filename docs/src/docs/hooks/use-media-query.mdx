# use-media-query

<PageHeader

use-media-query hook allows to subscribe to media queries. It receives media query as an argument and returns true if given media query matches current state.

## Usage

```tsx
import { useMediaQuery } from '@mantine/hooks';

function Example() {
  // matches will be true if (max-width: 765px) media query is satisfied
  const matches = useMediaQuery('(max-width: 765px)');
  return <div />;
}
```
