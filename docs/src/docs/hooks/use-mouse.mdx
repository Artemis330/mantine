## Usage

use-mouse returns moure x, y coordinates relatively to viewport screen:

<Demo data={useMouseUsage} />

## API

Hook returns object with ref and x, y mouse coordinates:

```tsx
const {
  ref, // -> pass ref to target element - optional in use
  x, // -> mouse x position
  y, // -> mouse y position
} = useMouse();
```

On the first render (as well as during SSR), both `x` and `y` values are equal to `0`.

## Examples

### Custom element

use-mouse handles move behavior over any element:

<Demo data={useMouseRef} />


## Definition

```tsx
function useMouse<T extends HTMLElement = any>(): {
  ref: React.MutableRefObject<T>;
  readonly x: number;
  readonly y: number;
};
```
