# use-id

use-id hook generates random id that persists across renders.
It is used to bind input elements to labels.

## Usage

```tsx
import { useId } from '@mantine/hooks';

// returns given id -> email
const providedId = useId('email');

// generates random id if id was not provided â€“> mantine-j_YSXk1CdUip5gPldwCjy
const randomId = useId();
```

## Example

```tsx
import React from 'react';
import { useId } from '@mantine/hooks';

function Example({ id }) {
  const uuid = useId(id);

  return (
    <>
      <label htmlFor={uuid}>Input label</label>
      <input type="text" id={uuid} />
    </>
  );
}
```
