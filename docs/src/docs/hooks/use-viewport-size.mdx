## Usage

Hook returns current viewport width and height:

<Demo data={useViewportSizeDemo} />

On the first render (as well as during SSR), both width and height are equal to `0`.

## Definition

```tsx
function useViewportSize(): Readonly<{
  height: number;
  width: number;
}>;
```
