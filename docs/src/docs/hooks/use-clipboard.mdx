# use-clipboard

use-clipboard hook provides interface to work with `navigator.clipboard`

## Import

```tsx
import { useClipboard } from '@mantine/hooks';
```

## Usage

```tsx
import React from 'react';
import { useClipboard } from '@mantine/hooks';

function Example() {
  const clipboard = useClipboard();

  return (
    <button type="button" onClick={() => clipboard.copy('Hello, world!')}>
      {clipboard.copied ? 'Copied' : 'Copy to clipboard'}
    </button>
  );
}
```

Clipboard object contains properties:

- `copy` – handler to copy given value to clipboard
- `copied` – boolean value that indicates that copy handler was recently called
- `reset` – handler to reset `copied` to false, by default reset will be automatically called in 2000 ms after `copy` handler
- `error` – contains Error if anything goes wrong

## Customize reset timeout

By default `copied` is reset to false in 2000 ms, this timeout can be changed by providing argument to hook call.

```tsx
const oneSecondTimeoutClipboard = useClipboard({ timeout: 1000 });
```
