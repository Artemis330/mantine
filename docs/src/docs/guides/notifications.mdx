## Installation

With yarn:

```sh
yarn add react-jss @mantine/theme @mantine/hooks @mantine/core @mantine/notifications
```

With npm:

```sh
npm install react-jss @mantine/theme @mantine/hooks @mantine/core @mantine/notifications
```

With [peerdeps](https://www.npmjs.com/package/install-peerdeps):

```sh
npx install-peerdeps @mantine/notifications
```

## Usage

Wrap your application with NotificationsProvider:

```tsx
import React from 'react';
import { NotificationsProvider } from '@mantine/notifications';

function YourApp() {
  return (
    <NotificationsProvider>
      <App />
    </NotificationsProvider>
  );
}
```

Use use-notifications hook at any place in your application:
