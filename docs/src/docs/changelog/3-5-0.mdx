## @mantine/prism changes

[Prism](https://mantine.dev/others/prism/) component now uses [ScrollArea](https://mantine.dev/core/scroll-area/) instead of native browser scrollbars:

<Demo data={PrismDemos.scrollbars} demoProps={{ toggle: true }} />

[Prism](https://mantine.dev/others/prism/) now supports organizing code with tabs:

<Demo data={PrismDemos.tabs} demoProps={{ toggle: true }} />

## NumberInput component improvements

[NumberInput](https://mantine.dev/core/number-input/) component now supports the following new features:

- decimal separator configuration
- value increment/decrement when user clicks and holds up/down controls

<Demo data={NumberInputDemos.decimalSeparator} />

<Demo data={NumberInputDemos.hold} />

## New features

[RingProgress](https://mantine.dev/core/ring-progress/) component now supports round caps:

<Demo data={RingProgressDemos.configurator} />

[Stepper](https://mantine.dev/core/stepper/) component now supports `Stepper.Completed` to display content after final step is completed:

<Demo data={StepperDemos.usage} />

## Other changes

- [Alert](https://mantine.dev/core/alert/) component now supports `radius` prop
- [Center](https://mantine.dev/core/center/) component is now polymorphic, it supports `component` prop
- [Grid](https://mantine.dev/core/grid/) component now exposes `Col` component as `Grid.Col`
- `@mantine/core` package now exports `MantineProviderProps` type

## 3.4.0 - 3.5.0 bug fixes

- Fix incorrect animations applied to [Collapse](https://mantine.dev/core/collapse/) component with `animateOpacity` prop set to false
- Fix [HueSlider and AlphaSlider](https://mantine.dev/core/color-picker/) components breaking when used without `size` prop
- Fix `size` prop not applied to [Divider](https://mantine.dev/core/divider/) with label
