# Mantine tag picker

[Notion](https://www.notion.so/) style tag picker with full keyboard support.

## Installation

With yarn:

```sh
yarn add react-jss @mantine/theme @mantine/hooks @mantine/core @mantine/tag-picker
```

With npm:

```sh
npm install react-jss @mantine/theme @mantine/hooks @mantine/core @mantine/tag-picker
```

With [peerdeps](https://www.npmjs.com/package/install-peerdeps):

```sh
npx install-peerdeps @mantine/tag-picker
```

## Multiple state

Multiple state is not currently supported, feel free to add this feature and add <GatsbyLink to="/pages/contribute/">open pull request</GatsbyLink>.
