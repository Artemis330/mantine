## Installation

Package does not depend on other `@mantine/` scoped packages and does not have other dependencies:

Install with npm:

```sh
npm install react-jss @mantine/notifications @mantine/core @mantine/hooks
```

Install with yarn:

```sh
yarn add react-jss @mantine/notifications @mantine/core @mantine/hooks
```
