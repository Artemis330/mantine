---
title: 'HoverCard component error'
error: 'HoverCard component was not found in the tree'
---

## Error source

You have rendered `HoverCard.Target` or `HoverCard.Dropdown` component outside of [HoverCard](/core/hover-card/) component context:

```tsx
import { HoverCard } from '@mantine/core';

function DemoWithError() {
  return (
    <>
      <HoverCard.Target>
        <button>Target</button>
      </HoverCard.Target>

      <HoverCard.Dropdown>Dropdown</HoverCard.Dropdown>
    </>
  );
}
```

## How to fix

Make sure that `HoverCard.Target` and `HoverCard.Dropdown` components are rendered within [HoverCard](/core/hover-card/) component
â€“ all of them require HoverCard context to work correctly:

```tsx
import { HoverCard } from '@mantine/core';

function DemoWithError() {
  return (
    <HoverCard>
      <HoverCard.Target>
        <button>Target</button>
      </HoverCard.Target>

      <HoverCard.Dropdown>Dropdown</HoverCard.Dropdown>
    </HoverCard>
  );
}
```
