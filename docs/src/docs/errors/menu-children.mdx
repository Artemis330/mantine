---
title: 'Menu component error'
error: 'Menu.Target component children should be an element or a component that accepts ref, fragments, strings, numbers and other primitive values are not supported'
---

## Error source

[Menu.Target](/core/menu/) was used with invalid children:

```tsx
import { Menu } from '@mantine/core';

function DemoWithError() {
  return (
    <>
      <Menu.Target label="Throws">Raw string, NOT OK – will throw error</Menu.Target>
      <Menu.Target label="Throws">{2}</Menu.Target>

      <Menu.Target label="Throws">
        <>Fragment, NOT OK, will throw error</>
      </Menu.Target>

      <Menu.Target label="Throws">
        <div>More that one node</div>
        <div>NOT OK, will throw error</div>
      </Menu.Target>
    </>
  );
}
```

## How to fix

Menu.Target children should be either an element or component that supports ref:

```tsx
import { Menu, Button } from '@mantine/core';

function Demo() {
  return (
    <>
      <Menu.Target label="OK">
        <button>Native button – ok</button>
      </Menu.Target>

      <Menu.Target label="OK">
        <Button>Mantine component – ok</Button>
      </Menu.Target>
    </>
  );
}
```
