---
title: 'HoverCard component error'
error: 'HoverCard.Target component children should be an element or a component that accepts ref, fragments, strings, numbers and other primitive values are not supported'
---

## Error source

[HoverCard.Target](/core/hover-card/) was used with invalid children:

```tsx
import { HoverCard } from '@mantine/core';

function DemoWithError() {
  return (
    <>
      <HoverCard.Target label="Throws">Raw string, NOT OK – will throw error</HoverCard.Target>
      <HoverCard.Target label="Throws">{2}</HoverCard.Target>

      <HoverCard.Target label="Throws">
        <>Fragment, NOT OK, will throw error</>
      </HoverCard.Target>

      <HoverCard.Target label="Throws">
        <div>More that one node</div>
        <div>NOT OK, will throw error</div>
      </HoverCard.Target>
    </>
  );
}
```

## How to fix

HoverCard.Target children should be either an element or component that supports ref:

```tsx
import { HoverCard, Badge } from '@mantine/core';

function Demo() {
  return (
    <>
      <HoverCard.Target label="OK">
        <button>Native button – ok</button>
      </HoverCard.Target>

      <HoverCard.Target label="OK">
        <Badge>Mantine component – ok</Badge>
      </HoverCard.Target>
    </>
  );
}
```
