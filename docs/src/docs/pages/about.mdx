# About Mantine

Mantine is a React components library focused on providing great user and developer experience.
Mantine development was started in January 2021 and 1.0 version was published on [May 3rd, 2021](https://github.com/mantinedev/mantine/releases/tag/1.0.0),
and since then [more than 100 releases were published](https://github.com/mantinedev/mantine/releases).
