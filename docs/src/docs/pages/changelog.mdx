# Changelog

## 1.3.0 – July 2021

### New components and features

<GatsbyLink to="/core/ring-progress/">RingProgress</GatsbyLink> component displays simple circle diagrams:
<Demo data={RingProgressDemos.usage} demoProps={{ toggle: true }} />

<GatsbyLink to="/core/tabs/">Tabs</GatsbyLink> component now supports outline variant:
<Demo data={TabsDemos.outline} demoProps={{ toggle: true }} />

<GatsbyLink to="/core/badge/">Badge</GatsbyLink> component now supports dot variant:
<Demo data={BadgeDemos.dots} />

<GatsbyLink to="/core/center/">Center</GatsbyLink> centers content vertically and horizontally:
<Demo data={CenterDemos.usage} demoProps={{ toggle: true }} />

### Other changes

- Mantine packages are now published with commonjs and es modules – this enables better tree shaking
- All components props types are now exported from @mantine/core package, you can import it by adding `Props` to component name:

```tsx
import { ButtonProps } from '@mantine/core';
```

### New components and features

### API Changes

- Prop typo fix in <GatsbyLink to="/core/checkbox/">Checkbox</GatsbyLink> component (~~intermediate~~
  -> indeterminate)
- Hr component replaced with <GatsbyLink to="/core/divider/">Divider</GatsbyLink> to support more options
- ElementsGroup component replaced with <GatsbyLink to="/core/group/">Group</GatsbyLink> to support all elements positioning
- Custom component is now supported by <GatsbyLink to="/core/action-icon/">ActionIcon</GatsbyLink> and <GatsbyLink to="/core/menu">MenuItem</GatsbyLink> components
- <GatsbyLink to="/core/modal/">Modal</GatsbyLink> and <GatsbyLink to="/core/drawer/">
    Drawer
  </GatsbyLink> components are wrapped with <GatsbyLink to="/core/portal/">Portal</GatsbyLink> by default
- <GatsbyLink to="/core/loading-overlay">LoadingOverlay</GatsbyLink> now supports custom loaders

### Other changes

- Significant performance improvements with react-jss useStyles props memoization
- Dark theme improvements, all components now use more balanced transparent colors in dark theme
